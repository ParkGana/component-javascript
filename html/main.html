BUTTON 컴포넌트는 a, button, div 등의 태그를 사용하여 생성할 수 있다.

  • a 태그는 페이지 이동 및 링크 연결이 필요한 경우 사용
  • button 태그는 페이지 내에서의 인터페이스 조작이 필요한 경우 사용
  • div 태그는 스타일적인 커스터마이징이 필요한 경우 사용
